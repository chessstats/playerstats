usage:

##################################

see also: https://github.com/chessstats/players
see also: https://github.com/chessstats/procplayers

collect keys from players.txt with procplayers

then at the R console type:

library(playerstats)
create_key_stats()

this will create statistical files for all collected keys in the 'current/stats' directory
